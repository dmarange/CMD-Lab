# Logging In To A Alliance Canada Cluster

The following screenshots were taken using Git Bash, but the process is the same across all terminal applications.

- In your terminal window, type the following command:

```bash
ssh -X USERNAME@CLUSTER.alliancecan.ca
```
- Enter your Alliance Canada account password when prompted and get your two-factor application available to verify that you are trying to log in:

- Your terminal will indicate whether the login was successful and may display additional messages or updates from Alliance Canada.
